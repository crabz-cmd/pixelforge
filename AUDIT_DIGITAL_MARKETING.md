## 🎯 RECOMMANDATIONS PRIORITAIRES (Par ordre d'urgence)

### 🔴 URGENCE CRITIQUE (Impact immédiat)

**1. Installer Google Analytics 4** — Priorité absolue
- Créer un compte GA4
- Installer le code de tracking
- Configurer les événements : formulaire soumission, clic CTA, modal ouverture
- Configurer les goals : formulaire contact, demande de devis
- **Impact** : Visibilité sur le trafic et les conversions
- **Complexité** : Faible
- **Temps** : 1-2 heures

**2. Ajouter les meta tags Open Graph et Twitter Cards** — Critique
- Ajouter `og:title`, `og:description`, `og:image`, `og:url`
- Ajouter `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image`
- Créer une image OG de 1200x630px
- **Impact** : Meilleur partage sur les réseaux sociaux
- **Complexité** : Faible
- **Temps** : 30 minutes

**3. Optimiser les images** — Critique pour la performance
- Compresser le logo (2.1 MB → <50KB)
- Convertir toutes les images en WebP
- Compresser les images projets (700KB-1.9MB → <100KB chacune)
- Ajouter `loading="lazy"` sur les images en-dessus du fold
- Ajouter les dimensions width/height pour éviter CLS
- **Impact** : Score Lighthouse 90+, temps de chargement -50%
- **Complexité** : Moyenne
- **Temps** : 2-3 heures

**4. Créer sitemap.xml et robots.txt** — Important pour le SEO
- Créer `sitemap.xml` avec toutes les URLs (incluant les pages légales)
- Créer `robots.txt` pour guider les crawlers
- Soumettre le sitemap à Google Search Console
- **Impact** : Indexation correcte du site
- **Complexité** : Faible
- **Temps** : 30 minutes

**5. Configurer Google Search Console** — Critique
- Créer un compte GSC
- Vérifier la propriété du site
- Soumettre le sitemap.xml
- Configurer les alertes de couverture
- **Impact** : Monitoring des erreurs SEO et des performances
- **Complexité** : Faible
- **Temps** : 1 heure

---
