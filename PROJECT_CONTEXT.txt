[8] DECISIONS IMPORTANTES
--------------------------------------------------------------------------------
- Architecture multi-page : index.html (main) + projects.html + FAQ.html + pages legales
- Tout CSS/JS inline dans chaque fichier pour simplifier le deploiement
- Zero dependance : aucune librairie externe, tout est vanilla
- Selecteur #nav (ID) au lieu de nav (tag) : critique pour eviter de cibler
  le <nav class="footer-nav"> dans le footer
- overflow:visible sur #nav : le logo depasse intentionnellement la hauteur
  de la navbar
- clip-path au lieu de mask-image pour le slider avant/apres : meilleure
  compatibilite cross-browser
- Fleches modale HORS du modal-box : evite le clipping par overflow:hidden
- scroll-behavior:auto sur html : le scroll est gere en JS custom pour
  permettre les effets de vitesse
- isolation:isolate sur #nav et body : empeche les canvas de "saigner"
  a travers la navbar
- transform:translateZ(0) sur canvas : optimisation GPU pour éviter flickering
